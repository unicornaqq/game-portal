### 今日题目：27. 移除元素 (双指针)

**核心思路：快慢指针法**

快指针 fast 遍历数组，当 `nums[fast] != val` 时，将值赋给慢指针 slow 位置，然后 slow++。最后返回 slow 即为新数组长度。

**Python 代码：**
```python
def removeElement(nums, val):
    slow = 0
    for fast in range(len(nums)):
        if nums[fast] != val:
            nums[slow] = nums[fast]
            slow += 1
    return slow
```

**复杂度：**
- 时间：O(n)
- 空间：O(1)

**面试要点：**
- 考察双指针思想、原地修改数组
- 保持相对顺序的处理方式
